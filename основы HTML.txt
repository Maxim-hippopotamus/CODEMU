ТЕГИ :

Теги - специальные команды для браузера :
Пример тега : начало <p> и конец </p>

АТРИБУТЫ :

Атрибуты - команды, расширяющие действие тега.
Атрибут размещается внутри тега : <тег атрибут1="значение" атрибут2="значение">

С помощью HTML создается архитектура сайта, а с помощью CSS эта структура украшается и размещается.
HTML +  CSS = верстка (размещение элеметов сайта по нужным местам) страниц.

ОСНОВНАЯ СТРУКТУРА СТРАНИЦЫ HTML :

Страница сайта - txt-файл с расширением .html

Этот файл должен иметь теги :
<html> (содержит в себе текст всего сайта)
Внутри него :
ВОПРОС - <head> (служебное содержание)
<body> (для основого текста, видного на экране)

В тег <head> входят :
<title> (задает название страницы)
ВОПРОС - <meta> (задает кодировку страницы, ставится в атрибуте charset и обычно имеет значение utf-8)

Перед тегом <html> обычно пишется конструкция doctype, которая указывает на версию языка HTML, на котором сделан сам сайт. Актуально сейчас - <!DOCTYPE html> .

ТЕГИ ДЛЯ АБЗАЦЕВ

<p> - перенос строки

ТЕГИ ДЛЯ ЗАГОЛОВКОВ :

<h1> , <h2> , <h3> , <h4> , >h5> , <h6> - заголовки, все жирные, отличаются размером

ТЕГИ ДЛЯ ВЫДЕЛЕНИЯ ЖИРНЫМ :

<b> - делает жирным

ТЕГИ ДЛЯ ВЫДЕЛЕНИЯ КУРСИВОМ :

<i> - курсив

НЕУПОРЯДОЧЕННЫЕ СПИСКИ :

В начале идет тег <ul>
 
<ul> - создает сам список

Потом идет тег <li>

<li> - создает черный кружок списка

УПОРЯДОЧЕННЫЕ СПИСКИ :

<ol> - создает сам список

<li> - создает номер списка

СОЗДАНИЕ ССЫЛОК :

<a> - создает ссылку

в самом теге <a> прописывается атрибут href, в который и записывается сама ссылка

Между <a> и </a> записывается название ссылки, как она будет отображаться на экране

Пример : <a href="https://www.google.com">link</a>

СОЗДАНИЕ ВНУТРЕННИХ ССЫЛОК (КОГДА ФАЙЛЫ ЛОКАЛЬНЫЕ)

Чтобы связать ссылками страницы своего сайта, нужно также прописать тег <a> и в нем же в атрибуте href прописать имена файлов, куда нужно сослаться :
Пример : <a href="page.html">link on Page</a> - page.html находится в корневой папке компа

ВСТАВКА КАРТИНОК :

<img> - вставляет картинку (не требует закрывающего тега), в нем же идет атрибут src, в который прописывается название картинки

ШИРИНА КАРТИНКИ И ВЫСОТА КАРТИНКИ :

<width> - задает ширину картинки, пропорции не меняет
<img src="smile.png" width="100">\

<height> - задает высоту картинки, пропорции не меняет
<img src="smile.png" height="100">

Также можно одновременно задать ширину и высоту :
<img src="smile.png" width="100" height="70">

Создание таблиц :

Пересечение столбца и строки - ячейка

<table> - создает таблицу, в нем может быть атрибут border (задает границу таблице и ее ячейкам) border="1"

Таблицы в HTML создаются по рядам - сначала идет первый ряд таблицы

<tr> - создает строки таблицы

Внутри прописываются ячейки :

<td> - создает ячейку в таблице

ШИРИНА И ВЫСОТА ТАБЛИЦЫ :

<table border="1" width="300" height="200">
